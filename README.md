# 🌦️ App del Clima

Aplicación web que permite consultar el clima en cualquier ciudad en tiempo real.

## 🚀 Tecnologías
- HTML
- CSS
- JavaScript
- API OpenWeather

## ✨ Funcionalidades
- Buscar ciudad
- Mostrar temperatura, clima y humedad
- Íconos dinámicos según el clima
- Cambio de fondo según condiciones
- Búsqueda con botón y tecla Enter
